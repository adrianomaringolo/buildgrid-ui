Sheet

View in Storybook

Explore interactive examples and all component variations in our Storybook.

O componente Sheet é um painel deslizante que aparece a partir das bordas da tela. É perfeito para menus de navegação, formulários, configurações ou qualquer conteúdo que precisa de mais espaço sem sair da página atual.

Importação​

import { 
  Sheet, 
  SheetContent, 
  SheetDescription, 
  SheetFooter, 
  SheetHeader, 
  SheetTitle, 
  SheetTrigger 
} from 'buildgrid-ui'

Uso Básico​

import { Sheet, SheetContent, SheetDescription, SheetFooter, SheetHeader, SheetTitle, SheetTrigger, Button } from 'buildgrid-ui'

function BasicSheet() {
  return (
    <Sheet>
      <SheetTrigger asChild>
        <Button>Abrir Sheet</Button>
      </SheetTrigger>
      <SheetContent>
        <SheetHeader>
          <SheetTitle>Título do Sheet</SheetTitle>
          <SheetDescription>
            Descrição do conteúdo que será exibido no sheet.
          </SheetDescription>
        </SheetHeader>
        <div className="py-4">
          <p>Conteúdo do sheet aqui.</p>
        </div>
      </SheetContent>
    </Sheet>
  )
}

API Reference​

Sheet​

PropTipoDescrição
defaultOpenbooleanEstado inicial do sheet (não controlado).
openbooleanEstado atual do sheet (controlado).
onOpenChange(open: boolean) => voidCallback chamado quando o estado muda.

SheetTrigger​

PropTipoDescrição
asChildbooleanSe true, renderiza como o elemento filho em vez de um botão.

SheetContent​

PropTipoDescrição
side"top" | "right" | "bottom" | "left"Lado da tela de onde o sheet aparece. Padrão: "right".
classNamestringClasses CSS adicionais.
